README:
This program is the main core of the Integrated Control Environment,
which is a general use process controller that is configured to
interface with numerous types of inputs and outputs ranging from
relays, actuators, temperature sensors, LCD panels, touch screens, etc.

This platform is still under development so it may not work as intended.
------------------------------------------------------------------------
